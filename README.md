# sp6-2_smart-table_starter

Современная, кастомизируемая таблица данных с расширенными возможностями сортировки, фильтрации, поиска и пагинации. Построена на **Vite** для максимальной производительности.

## ✨ Возможности

- 🔍 **Глобальный поиск** по всем колонкам
- ⬆️ **Сортировка** по клику на заголовок
- 🎯 **Фильтрация** данных по условиям
- 📄 **Пагинация** для работы с большими объемами
- 🎨 **Кастомизация** под любые задачи
- 📱 **Адаптивный дизайн**
- 🚀 **Быстрая работа** благодаря Vite

## 🛠️ Технологии

- **Vite** - сборщик проекта и dev-сервер
- **Vanilla JavaScript** - чистая логика без фреймворков
- **CSS3** - стилизация с современными подходами
- **Yandex Sans (YS Text)** - профессиональные шрифты
